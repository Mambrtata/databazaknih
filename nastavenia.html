Uložené.

Jazyk knižnice

Nové popisy kníh sa pri dopĺňaní automaticky preložia do tohto jazyka — vyžaduje Gemini API kľúč nižšie.

API kľúče

Bez kľúča má Google Books API veľmi nízky a nestabilný limit (časté chyby 429). Môžeš použiť ten istý kľúč ako pre Gemini, ale v Google Cloud Console mu musíš najprv povoliť „Books API".

Kľúč sa ukladá len v tomto prehliadači (localStorage), nikam sa neposiela okrem priamo do Google Gemini API. Získaš ho na aistudio.google.com/apikey.

Zdieľanie knižnice

Zapni, ak chceš niekomu poslať odkaz na prezeranie tvojej knižnice (bez možnosti upravovať, bez prihlásenia).

Záloha katalógu

JSON = plná záloha (vrátane obalov), pre prenos medzi tvojimi zariadeniami. CSV = zoznam pre import do Goodreads/LibraryThing (bez obalov).

⚠️ Nebezpečná zóna

Tieto akcie sú nezvratné.

Vymaže celý tvoj katalóg (knihy), účet a prihlásenie zostanú — môžeš začať znova od nuly.

Úplne zmaže tvoj účet (vrátane prihlasovacích údajov) aj všetky knihy. Po tomto kroku sa už nebudeš môcť prihlásiť pod týmto účtom.
